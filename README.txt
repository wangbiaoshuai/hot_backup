说明：
1.主备程序占用8101端口。
2.主备程序之间的心跳为5S。
3.主程序每隔30S检测主机状态（monitor进程和mysql）。
4.主备程序安装在/usr/local目录下。
5.在主服务器被替换之后，主服务器的IP将会被切换到本网段的一个未被占用的IP上。备服务器将被切换到主服务器的IP上。

部署方法：
1.主备程序需要运行在两台不用的机器上，两台机器可以ping通，两台机器必须要有自己的静态IP。
2.需要保证在mysql和monitor已经启动的情况下启动主程序。(sh install.sh)
3.需要在主程序运行起来的情况下运行备程序。(sh install.sh server_ip) server_ip为主程序所在的ip地址。
